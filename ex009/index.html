Trabalhando com listas

Listas ordenadas

Usar a tag <ol> com a tag <li> para cada item da lista. Usar type dentro da tag <ol> para definir o tipo de pontuação e start para definir o começo da pontuação. Ex:

  1. Acordar
  2. Escovar os dentes
  3. Estudar HTML e CSS

OBS: Start só funciona com type="1"

Listas não ordenadas

Usar a tag <ul> também com a tag <li> para cada item da lista. Também usar o type dentro de <ul> para modificar a marcação

  • pão
  • leite
  • arroz
  • feijão

Lista de definições

Usar <dl> para abrir a lista de definições, depois <dt> para o termo e <dd> para a definição.
