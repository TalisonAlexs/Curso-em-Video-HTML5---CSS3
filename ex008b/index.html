Outras formatações

Código-fonte

Usar a tag <code> para fonte monoespaçada (boa para linha de código). Ex: O comando document.getElementById('teste') é escrito em JavaScript

Usar a tag <pre> para preservar a formatação do texto. Utilizando <code> dentro de <pre> pode-se emular um código melhor. Ex: 

            
                n = int(input("Digite um número inteiro: "))
                if n % 2 == 0:
                    print(f"O número {n} é par")
                else:
                    print(f"O número {n} é ímpar")

Citações

Usar a tag <q> para colocar entre aspas (ideal para citações simples).Ex: Como dizia Schopenhauer: O Homem é livre para fazer o que quer, mas não para querer o que quer.

Usar a tag <blockquote> para colocar entre aspas (ideal para citações completas). Ex:

A fonte de onde emanam os indivíduos e as suas forças é inesgotável e infinita, tanto como o tempo e o espaço, visto que, como o tempo e o espaço, ela é apenas o fenômeno e a representação da vontade.

OBS: Usar cite = "url" dentro do <q> ou <blockquote> para citar a pagina de onde foi tirado a citação

Abreviações

Usar a tag <abbr> para abreviações e siglas. Ex: HTML, CSS, JS

Texto invertido

Usar a tag <bdo> para inverter o texto. Ex: texto invertido da esquerda para a direita. Texto invertido da direita para a esquerda
