Principais formatações

Negrito / Destaque

Usar a tag <b> para negrito (não semântica) ❌

Usar a tag <strong> para destaque (semântica) ✔

Itálico / Enfase

Usar a tag <i> para itálico (não semântico) ❌

Usar a tag <em> para enfase (semântico) ✔

Texto marcado

Usar a tag <mark> para texto marcado

Utilizar <style> no head para estilizar todos os marcadores ou style dentro <mark> para estilizar individualmente os marcadores

Texto grande e pequeno

Usar a tag <big> para texto grande e a tag <small> para texto pequeno

OBS: A tag <big> esta obsoleta

Texto deletado

Usar a tag <del> para riscar um texto, indicando que ele deve ser lido mas não considerado

Texto incluído

Usar a tag <ins> para sublinhar um texto, indicando que ele foi adicionado depois

OBS: Também existe o sublinhado com a tag <u> que não é semântica

Texto sobrescrito

Usar a tag <sup>. Ex: x2+5

Texto subscrito

Usar a tag <sub>. Ex: H2O
